# Инфраструктура как код с использованием Terraform и Proxmox

## Введение
В современной практике системного администрирования и DevOps стремительно набирает популярность принцип «Инфраструктура как код» (IaC). Этот подход позволяет управлять вычислительными ресурсами не через ручные клики в графическом интерфейсе, а с помощью декларативных конфигурационных файлов, которые можно версионировать, тестировать и автоматически применять. В контексте виртуализации на базе Proxmox VE, мощной и гибкой платформы, реализация IaC становится ключом к созданию воспроизводимой, контролируемой и легко масштабируемой инфраструктуры. Инструментом, который идеально подходит для этой задачи, является Terraform от HashiCorp. Данная статья представляет собой практический опыт интеграции Terraform с Proxmox, описывая путь от начальной настройки до полного перехода на декларативное управление виртуальными машинами.

## Описание проекта
Управление виртуальной инфраструктурой Proxmox с помощью Terraform.
