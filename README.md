# Infrastructure as Code with Terraform and Proxmox

## Описание проекта
Проект демонстрирует управление виртуальной инфраструктурой Proxmox с помощью Terraform.

## Быстрый старт
1. Заполните `terraform.tfvars` реальными значениями
2. Выполните:
```bash
terraform init
terraform plan
terraform apply
